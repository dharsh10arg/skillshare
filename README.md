# SkillShare Mobile Repository

This repository contains the Flutter mobile app for SkillShare.

## Repository structure

- `frontend/` - Main Flutter application directory
  - `frontend/lib/` - Dart source files for the app
  - `frontend/pubspec.yaml` - Flutter project configuration and dependencies
  - `frontend/test/` - Widget tests for the mobile app

## Getting started

1. Install Flutter: https://flutter.dev/docs/get-started/install
2. Navigate to the frontend directory:

```bash
cd frontend
```

3. Get dependencies:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```

## Notes

- The Flutter project is intentionally stored under `frontend/`.
- This repo is linked to: `https://github.com/dharsh10arg/skillshare.git`
